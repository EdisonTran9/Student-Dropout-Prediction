Overview

This project analyzes student data to predict post-secondary dropout risk using machine learning models. The objective is to identify key factors contributing to student attrition and provide data-driven insights that can support early intervention strategies by educational institutions and policymakers.

Dataset

The dataset consists of 4,424 student records and includes a range of academic, demographic, and financial variables that may influence student outcomes.

Methods

The project involved data cleaning and preprocessing in Python, followed by exploratory data analysis to uncover patterns and relationships within the data. Feature importance analysis was conducted to determine which variables had the most significant impact on predicting dropout risk.

Models

Several machine learning models were implemented and compared, including Decision Tree, Random Forest, and Gradient Boosting algorithms, to evaluate their predictive performance.

Results

The Gradient Boosting model achieved the highest performance with an accuracy of 79.1%. The analysis revealed that tuition payment status and second-semester academic performance were among the most important predictors of student dropout.

Visualizations

Various visualizations were created to support the analysis, including correlation matrices, boxplots, and confusion matrices, helping to better understand the relationships between variables and model performance.

Tools and Technologies

The project was developed using Python, with key libraries including pandas for data manipulation, scikit-learn for machine learning, and matplotlib for data visualization.

Key Insights

The findings indicate that both financial and academic factors play a critical role in student retention. Identifying at-risk students early allows institutions to design targeted interventions that can reduce dropout rates and improve overall student success.
